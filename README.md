# Parrots Card Game

### How to play this game
This is a simple but funny game that runs on a web browser (chrome recommended), all you have to do is clone this repository and open the index.html file with a browser. Then you can pick the number of cards you want to play.

Like any memory game, you flip two cards, if they're equal they'll stay flipped, else, they will turn down again. Match all pairs to win the game.
